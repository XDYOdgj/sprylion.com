Lehoo Castle Toddler Utensils Set

This adorable utensils set is designed to make self-feeding fun and easy for little ones. With 3 spoons and 3 forks included, it's the perfect start for toddlers learning to eat independently. The playful dinosaur pattern adds a whimsical touch that will keep kids entertained while they're eating.
